# Color asset plugin for EasyLogic Studio 

This plugin makes color asset property.

## Features 

* run new component 
* add BaseProperty in library 

# Local Dev 

```
npm install
npm install -D @easylogic/editor @easylogic/color
npm run dev 
```

# How to run

```js
npm run dev 
```

# How to build 

```js
npm run build
```
